# Assignment_2
Program 1: Sum of Numbers from 1 to 50
File: task1_A2.py
Functionality
Calculates the sum of all integers from 1 to 50.
Uses a for loop to iterate through the range of numbers.
Stores the cumulative sum in a variable.
Displays the final sum as output.
Output
The program prints the total sum of numbers from 1 to 50.

Program 2: Even or Odd Number Checker
File: task2_A2.py
Functionality
Accepts an integer input from the user.
Determines whether the entered number is even or odd.
Uses the modulus operator (%) to perform the check.
Displays the result accordingly.
Output
The program prints whether the given number is even or odd.

Execution
Both programs can be executed using Python 3 from the command line:
python task1_A2.py
python task2_A2.py
